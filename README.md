# Usage
Currently only chrome browser is supported.

##### 1.
```
$ git clone https://github.com/jehyeon/patty.git
```
or [download .zip](https://github.com/jehyeon/patty/archive/master.zip)

##### 2.
Go to **chrome://extensions** and **Load unpacked**
